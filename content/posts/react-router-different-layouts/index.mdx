Layouts

```typescript
<Switch>
  <Route exact path={["/"]}>
    <DefaultLayout>
      <Switch>
        <Route
          exact
          path="/"
          component={lazy(() => import("../pages/Dashboard"))}
        />
      </Switch>
    </DefaultLayout>
  </Route>
  <Route path={["/signin", "/signup", "*"]}>
    <FullWidthLayout>
      <Switch>
        <Route
          path="/signin"
          component={lazy(() => import("../pages/User/SignInPage"))}
        />
        <Route
          path="/signup"
          component={lazy(() => import("../pages/User/SignUpPage"))}
        />
        <Route component={NotFound} />
      </Switch>
    </FullWidthLayout>
  </Route>
  <Route path={["*"]}>
    <FullWidthLayout>
      <Switch>
        <Route component={NotFound} />
      </Switch>
    </FullWidthLayout>
  </Route>
</Switch>
```
